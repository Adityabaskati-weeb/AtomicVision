## Sample Inputs

These files are lightweight example spectra for the AtomicVision demo UI.

- `sample-medium.csv`
- `sample-medium.txt`
- `sample-medium.json`

They are intended as upload examples for the Hugging Face Space and GitHub repo.
At the moment, the premium frontend accepts these formats as part of the demo
experience, while the backend analysis still runs against the live OpenEnv
episode flow.
